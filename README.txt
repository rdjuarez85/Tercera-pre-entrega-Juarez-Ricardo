|       -- Aplicación que muestra la funcionalidad de una empresa que brinda soluciones electrónicas                                        |
|                                                                                                                                           |
|       -- Consta de 3 modelos que son:                                                                                                     |
|                                       - Cliente                                                                                           |
|                                       - Técnico                                                                                           |
|                                       - Trabajo                                                                                           |
|                                                                                                                                           |
|       -- En esta aplicación es posible ingresar los datos de un nuevo cliente, de un nuevo técnico y de ordenes de trabajo                |
|                                                                                                                                           |
|       -- También es posible realizar una búsqueda en la base de datos de cada uno de los modelos                                          |
|           - En los modelos Cliente y Técnico va a buscar los parámetros ingresados que coincidan en los campos Nombre y/o Apellido        |
|           - En el modelo Trabajo va a buscar los parámetros ingresados que coincidan en los campos Dispositivo, Falla y/o Estado          |
|                                                                                                                                           |
|       -- El orden de prueba es indistinto ya que consta con algunos datos guardados en la base de datos                                   |
|                                                                                                                                           |
|       -- Para navegar entre Clientes, Técnicos y Ordenes de trabajo se deberá volver a la página de inicio haciendo click en "Home"       |
|                                                                                                                                           |
|       -- Los links a "Nosotros" y "Contacto" tambien fucionan                                                                             |
